Rules: Main object of the game is kick all numbers with the same values by clicking on them. Only nine numbers must left from 1 to 9 in any place on the field.
Two numbers can be kicked only if:

  1. their values are the same
  2. numbers in the same column or row
  3. you can click on two numbers also if there is other numbers between them
  4. If there is numbers between, they can be kicked only if all other numbers between was kicked
